# narou-translator — 일본 웹소설 번역 Chrome 확장

일본 웹소설 사이트 '소설가가 되자(小説家になろう)'를 한국어로 번역해 읽는 Chrome 확장 프로그램.

## 특징
- 페이지 내 본문 자동 감지 → Gemini 번역 → 원문 자리에 표시
- **작품별 용어집(glossary)** — 고유명사·인명 번역을 작품 단위로 고정해 일관성 유지
- 옵션 페이지에서 API 키·번역 스타일 설정

## 스택
JavaScript · Chrome Extension(Manifest V3) · Google Gemini API
